# Bug Report

## BUG001 - Login permite acesso sem senha

### Descrição
Sistema permite login mesmo sem senha preenchida.

### Passos para reproduzir:
1. Acessar login
2. Inserir usuário válido
3. Deixar senha vazia
4. Clicar em login

### Resultado atual:
Sistema permite acesso

### Resultado esperado:
Bloquear login e exibir mensagem de erro

### Severidade:
Alta (Critical)
